消息队列
RabbitMQ 本身支持很多的协议：AMQP，XMPP, SMTP, STOMP，也正是如此，使的它变的非常重量级，更适合于企业级的开发
ZeroMQ 号称最快的消息队列系统，尤其针对大吞吐量的需求场景。ZMQ能够实现RabbitMQ不擅长的高级/复杂的队列。
ActiveMQ 是Apache下的一个子项目。 类似于ZeroMQ，它能够以代理人和点对点的技术实现队列
Kafka 是一个高性能跨语言分布式Publish/Subscribe消息队列系统
